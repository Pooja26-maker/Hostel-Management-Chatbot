# 🏨 Hostel Management Chatbot

An AI-based chatbot developed to assist students with hostel-related queries such as room availability, hostel rules, fee information, facilities, and complaint handling.

## Features

- Answer hostel-related questions
- Provide room and facility information
- Hostel rules and regulations support
- Student query assistance
- Interactive conversational interface

## Technologies Used

- Python
- Google Colab
- NLP
- Machine Learning

## Future Enhancements

- Database Integration
- Voice Assistant Support
- Multi-language Support
- Web Deployment
